P2 - Frogment

Frogment - Project Goal

The goal of this project was to create a puzzle mobile game inspired after Monument Valley. The game had to be playable on most mobile devices, even the weaker ones.

Frogment Overview

Specifications:
- Engine: Unity
- Platform: Mobile
- My Role: Envirmental Art, Technical Art

The game features four distinct levels, each with each featuring it's own puzzles and unique enviroments. Play as as a magical frog traversing through levels and solving puzzles to catch Queen Firefly.

My role for the project was to create shaders and particle system used to enchance enviroments. Additionally I was helping graphic designers by producing enviromental 3D models such as trees, mountains, crystals and rocks.

Featured work - Wter Shader

My featured work in this project was a water shader. The goal of this work was to create a universal shader that could be used to create distinct and fitting variants of water effects for different levels. For that puprose the shader featured a lot of different parameters that could have been used by other discplines to create and effect they desire.

Shader would be applied on a plane that was spread across whole level or used locally. It used Unity's procedural noises to create effects such as water foam, floating cherry blossom flowers, depth effect and waves.
