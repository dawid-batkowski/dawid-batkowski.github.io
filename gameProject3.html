P3 - The Misadventures of Captain Hornswaggle

TMOCH - Project Goal

The goal of this project was to create 2.5D platformer game on a custom engine.

TMOCH Overview

Specifications:
- Engine: TGE (Custom)
- Platform: PC
- My Role: Technical Art, Animator

First group project in a custom engine. The game features four distinct levels. Each levels has different obstacles player has to overcome.

My role for the project was to create shaders, flipbook VFX, rigs and animations. This was not only my first project in a custom engine but it was also first encounter with HLSL.

Featured work - Cel Shader

My featured work in this project was a cel shader master material. As a group we decided to feature in our game very cartoony style, with cel shading, outlines, exaggurated animations, disproprtional character models and vibrant enviroments. The cel shader was a result of cooperation with second technical artist from my group Efrem Torrisi. Shader was modyfing DirectionalLight information, quantazing it and remaping it's values until desired result from reference was achieved. Later on, a sharp specualr was added to further enchance stylized look.

Additional work - Rigs and animations

One of my tasks during this project was to create rigs and animations for main player character and parrot enemy character. All arigs and animations were created in blender. Squash and stretch techqnique was commonly used during creation process of these animations to better fit into cartoony style.

Additional work - VFX and shaders

Background fog was created using scene world position to hide sharp transition between edge of water plane and background.

Death VFX for player character was created using Unreal's Niagara particle system. Each particle was using a fbx created with help of meta balls in blender with applied cel shading similar to one in our game. The particles were spawned using vertex position of a skull FBX created in blender. Effect then was baked into flipbook and replayed in game using flipbook player.

Water shader was created using a combination of 2D gradient and simple noises to create main water pattern. Additionally a surface foam was added by adding finner gradient noise. Shader was then applied on a subdivided plane and a simple vertex shader was written to create subtle displacement.

Crate destruction simulation was created using Houdini. Crate fbx was imported into houdini, there using material fracture and rbd bullet solver it was fractured and simulated. Then the bones were added to the pivot point of each piece and whole thing was exported. In blender additional cleaning, bone assigning and small animation tweaking was needed before rigged mesh and animation can be imported into our engine.
