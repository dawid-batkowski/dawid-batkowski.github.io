P4 - Cannibal Crossing

Cannibal Crossing - Project Goal

The goal of this project was to create a 3D top-down adventure game. The game had to feature multiple different levels, two enemy types and elite enemy varaint, feeling of progression and needed to have clear player motivation.

Cannibal Crossing - Overview

Specifications:
- Engine: TGE (Custom DX11)
- Platform: PC
- My Role: Technical Art, Animator

Second group project in TGE. The game features four distinct levels. Each levels has different obstacles player has to overcome.

My role for the project was to create shaders, VFX, rigs and animations. I was specifically tasked with riging enemy characters, creating envirmental shaders such as fog, water.

Featured work - Triplanar Shader

My featured work in this project was a triplanar shader for tillable textures. Main function uses vertex normal information from the vertex shader to determine which wolrd coordinates are gonna be used for tilling the texture. After that, the texture is sampled 3 times, each time with different coordinates and lerped with eachother using normal data as a t value of the lerp. The smoothness of blending between textures and tilling density can be controlled for each case separately.

Shader can be applied on vareity of shapes other than cubes, it works with non-uniform scaling and allows for rotation.

Additional work - Rigs and Destruction sims

Creating rigs for melee and ranged enemies was one of my main tasks. Those rigs would then be used by Graphic designers to create animations.

Destructable objects was another requirement for this project. For that Houdini fracturing nodes and rbd bullet solver were used to simulate the destruction. Then bones were assigned to pivot point of each piece and whole thing was then exported, imported into blender, cleaned and prepared for importing into our engine.

Additional work - VFX and shaders

Fog planes were created to be placed around level. Shader was using height calculation between plane with shader and scene height. Additionally, a panning noise was added to create less uniform and more dynamic effect.

On hit effect for enemy and player was created using simple Fresnel function in combination with constant buffer variables to time it properly.

Current health indicator was created for characters. It used multi-layered nosie textures in combination with health constant buffer value to make both player and enemy characters more bloody the lower health they have.

Fire shader for campfires and braziers was created by combining vertex shader displacement, panning noise for coloring and height alpha clipping. Alpha clipping was used instead of alpha or additive blend as it was less prone to encounter sorting issues and fit the stylized look of the game.

Water shader was created in similar fashion to fog planes by comparing depths to create foam and water depth. Noise texture created in Substance Designer was used create fake waves. Additionally, shadow map distortion was introduced to further sell the fake wave effect.

Cloud shader was a simple but noticable addition to out project. The shader was making use of cast shadows already present in our game at the time and the fact that camera could never look up. By streatching a large plane above the whole level and using a simple panning noise shader on the plane with fairly agressive alpha clipping an illusion of clouds casting shadows was created.

Post processing effects were done in the fullscreen pass to further enchance visuals of our game. These effects mainly included vignette and color adjustments.
